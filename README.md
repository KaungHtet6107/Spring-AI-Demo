# 🤖 Spring AI Demo (Full Stack)

This is a full-stack AI-powered web application built with Java Spring Boot (Backend) and React (Frontend).
The application integrates AI features such as:

🖼️ Image Generator
💬 Ask AI (Chat)
🍳 Recipe Generator

---

📸 Features

🖼️ Image Generator
Generate AI images based on user prompts
Simple UI with preview grid

💬 Ask AI
Chat with AI using natural language
Displays responses dynamically

🍳 Recipe Generator
Generate recipes based on ingredients
Supports dietary restrictions

---
